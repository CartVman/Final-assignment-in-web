Welcome to the "My Playlist" page. This page is part of the project for our final assignment in web development. We use only HTML and CSS to build this page. It is important to note that this page's contents are just made up and based on imagination.

About the Page

This page contains information about George Washington's four favorite playlists. The charts can be hovered and will provide extra information. The page is accessible for screen reader by pressing the "Tab" key on the keyboard.
